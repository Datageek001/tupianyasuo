# 图片压缩工具

一个简单易用的在线图片压缩工具，具有现代化的界面设计。用户可以通过拖放或点击上传图片，预览压缩效果，并下载压缩后的图片。

## 功能特点

- 支持 PNG、JPG 格式图片上传
- 拖放上传支持
- 实时预览压缩前后的效果
- 可调节压缩质量（1-100%）
- 显示原始和压缩后的文件大小
- 一键下载压缩后的图片
- 响应式设计，支持各种设备

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Canvas API

## 本地运行

1. 克隆仓库：
   ```bash
   git clone https://github.com/Datageek001/PictureCompress.git
   ```

2. 进入项目目录：
   ```bash
   cd PictureCompress
   ```

3. 使用浏览器打开 index.html 文件

## 使用说明

1. 点击上传区域或将图片拖放到指定区域
2. 等待图片加载完成
3. 使用质量滑块调节压缩程度
4. 点击"下载压缩后的图片"按钮保存
